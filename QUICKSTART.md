# 快速开始指南

## 第一次使用

### 1. 安装 MongoDB (如果尚未安装)

**Windows 用户:**
1. 访问 https://www.mongodb.com/try/download/community
2. 下载 MongoDB Community Server (Windows MSI)
3. 运行安装程序,选择 "Complete" 安装
4. 安装完成后,创建数据目录:
   ```
   mkdir C:\data\db
   ```

### 2. 启动应用

#### 方法一: 使用启动脚本 (推荐)
双击运行 `start.bat` 文件,脚本会自动:
- 检查 MongoDB 服务
- 检查项目依赖
- 启动前后端服务

#### 方法二: 手动启动

**步骤 1 - 启动 MongoDB:**
- 双击运行 `start-mongodb.bat`
- 或者以管理员身份运行命令提示符,执行: `net start MongoDB`

**步骤 2 - 打开新终端,启动应用:**
```bash
npm start
```

### 3. 访问应用
打开浏览器访问: http://localhost:3000

## 主要功能

### 生产账单信息 (左侧菜单第一项)

**创建订单:**
1. 点击 "新建订单" 按钮
2. 填写工厂名称、货物信息、生产总价等
3. 点击 "保存"

**添加材料费用:**
1. 展开某个订单
2. 在 "材料/其他费用" 区域,点击 "添加材料"
3. 选择类型(面料/五金/复合等)
4. 填写价格和备注
5. 点击 "保存"

**添加付款记录:**
1. 展开某个订单
2. 在 "付款记录" 区域,点击 "添加付款"
3. 填写付款对象和金额
4. 点击 "保存"

**查看应付金额:**
系统自动计算: 应付金额 = 生产总价 - 材料费用 - 已付款

**筛选和搜索:**
- 按工厂名称筛选
- 按材料类型筛选
- 关键词搜索(搜索任意字段)
- 按时间段筛选
- 可以同时使用多个筛选条件

**导出数据:**
- "导出全部" - 导出所有订单
- "导出筛选结果" - 仅导出当前筛选的订单

### 发货货款统计 (左侧菜单第二项)

**创建订单:**
1. 点击 "新建订单" 按钮
2. 填写订货方、货物信息、价格等
3. 货物图片可以粘贴图片URL
4. 点击 "保存"

**添加付款:**
1. 点击订单左侧的展开按钮
2. 在 "付款记录" 区域,点击 "添加付款"
3. 填写定金和已付金额(可以为负数)
4. 点击 "保存"

**查看欠款:**
系统自动计算: 欠款金额 = 应付总价 - 所有付款记录的总和

**筛选和搜索:**
- 按订货方筛选
- 按货品名称/货号筛选
- 按付款状态筛选(已付清/未付清)
- 关键词搜索
- 按时间段筛选

**导出数据:**
点击 "导出Excel" 按钮导出当前数据

## 常用操作技巧

### 1. 快速添加订单
连续添加多个订单时,保存后对话框会关闭,可以立即点击 "新建订单" 继续添加

### 2. 批量查看
使用筛选条件找到需要的订单,然后一次性导出到 Excel 进行打印或存档

### 3. 负数的使用
- 材料价格可以为负数(比如材料退款)
- 发货订单的已付金额可以为负数(比如退款)

### 4. 实时统计
顶部显示的总欠款会根据筛选条件实时更新,方便查看特定条件下的应付金额

### 5. 关键词筛选
如果要查看与某个材料厂相关的所有记录:
1. 在关键词搜索框输入材料厂名称
2. 点击查询
3. 系统会显示所有包含该关键词的订单

## 数据备份

建议定期备份 MongoDB 数据:

```bash
# 备份命令
mongodump --db order_management --out C:\backup\order_$(date +%Y%m%d)

# 恢复命令
mongorestore --db order_management C:\backup\order_20240101
```

## 问题排查

### 应用无法启动
1. 检查 MongoDB 是否运行
2. 检查端口 3000 和 5000 是否被占用
3. 查看终端错误信息

### 数据未保存
1. 确认 MongoDB 服务正常
2. 检查浏览器控制台是否有错误
3. 确认必填字段已填写

### 导出失败
1. 确认浏览器允许下载文件
2. 确认有订单数据
3. 尝试刷新页面后重试

## 技术支持

如遇到问题:
1. 查看 README.md 文件
2. 检查终端的错误信息
3. 确认 MongoDB 和 Node.js 版本是否符合要求
