# 功能使用说明

## 系统概览

订单管理系统分为两大模块:
1. **生产账单信息** - 管理与工厂的生产订单、材料费用和付款
2. **发货货款统计** - 管理向客户发货的订单和收款情况

---

## 一、生产账单信息模块

### 1.1 创建生产订单

**步骤:**
1. 点击页面顶部 "新建订单" 按钮
2. 填写表单:
   - **工厂名称*** (必填): 生产厂家的名称
   - **货物信息**: 生产的货物描述
   - **数量**: 订单数量
   - **单价**: 每件商品的价格
   - **生产总价*** (必填): 与工厂约定的总价格
3. 点击 "保存"

**示例:**
- 工厂名称: 宏达制衣厂
- 货物信息: T恤衫 - 白色
- 数量: 1000
- 单价: 15
- 生产总价: 15000

### 1.2 添加材料/其他费用

订单创建后,您可能需要支付材料费用给材料供应商。

**步骤:**
1. 点击订单左侧的展开按钮
2. 在 "材料/其他费用" 区域,点击 "添加材料"
3. 填写表单:
   - **类型*** (必填): 选择材料类型
     - 面料
     - 五金
     - 复合
     - 织袋
     - 塑料袋
     - 里布
     - 其他
   - **备注**: 详细说明,如供应商名称、材料详情
   - **价格*** (必填): 材料费用(可以为负数,表示退款)
   - **日期**: 支付日期
4. 点击 "保存"

**示例:**
- 类型: 面料
- 备注: 纯棉面料 - 信达布料厂
- 价格: 5000
- 日期: 2024-02-15

**特殊说明:**
- 可以添加多条材料记录
- 价格可以为负数(比如材料退款)
- 系统会自动从生产总价中扣除材料费用

### 1.3 添加付款记录

记录支付给工厂或材料厂的款项。

**步骤:**
1. 展开订单
2. 在 "付款记录" 区域,点击 "添加付款"
3. 填写表单:
   - **付款对象*** (必填): 收款方名称
   - **金额*** (必填): 付款金额
   - **备注**: 付款说明
   - **日期**: 付款日期
4. 点击 "保存"

**示例:**
- 付款对象: 宏达制衣厂
- 金额: 5000
- 备注: 预付款
- 日期: 2024-02-15

### 1.4 查看应付金额

系统自动计算应付金额:

```
应付金额 = 生产总价 - 材料费用总和 - 已付款总和
```

**示例计算:**
- 生产总价: ¥15,000
- 材料费用: ¥5,000 + ¥800 = ¥5,800
- 已付款: ¥5,000
- **应付金额: ¥4,200**

显示位置:
- 订单标题栏右侧(折叠状态)
- 订单详情中(展开状态)
- 页面顶部总欠款

### 1.5 筛选和搜索订单

**筛选条件:**
- **工厂名称**: 输入工厂名称筛选
- **材料类型**: 选择材料类型(如:面料、五金)
- **关键词搜索**: 搜索任意字段内容
- **时间范围**: 选择日期范围

**高级搜索示例:**

**场景1: 查找某个材料厂的所有订单**
- 关键词搜索: "信达"
- 点击 "查询"
- 结果: 所有材料备注中包含"信达"的订单

**场景2: 查看1月份所有订单**
- 时间范围: 2024-01-01 至 2024-01-31
- 点击 "查询"

**场景3: 查看某工厂的面料费用**
- 工厂名称: 宏达制衣厂
- 材料类型: 面料
- 点击 "查询"

**组合筛选:**
可以同时使用多个筛选条件,系统会显示同时满足所有条件的订单。

### 1.6 导出数据

**两种导出方式:**

**1. 导出全部**
- 点击 "导出全部" 按钮
- 导出数据库中的所有订单
- 不受当前筛选条件影响

**2. 导出筛选结果**
- 先设置筛选条件
- 点击 "导出筛选结果" 按钮
- 只导出当前显示的订单

**Excel 文件内容:**
- **订单汇总** 表: 所有订单的概览
- **详细信息** 表: 每个订单的完整信息
- **材料汇总** 表: 按材料类型统计

### 1.7 编辑和删除订单

**编辑订单基本信息:**
1. 展开订单
2. 点击 "编辑" 按钮
3. 修改信息后点击 "保存"

**删除材料记录:**
1. 展开订单
2. 在材料表格中找到要删除的记录
3. 点击 "删除" 按钮
4. 确认删除

**删除付款记录:**
1. 展开订单
2. 在付款表格中找到要删除的记录
3. 点击 "删除" 按钮
4. 确认删除

**删除整个订单:**
1. 展开订单
2. 点击 "删除" 按钮
3. 确认删除
4. **警告:** 此操作会删除订单及其所有材料和付款记录

## 二、发货货款统计模块

### 2.1 创建发货订单

**步骤:**
1. 切换到 "发货货款统计" 标签
2. 点击 "新建订单" 按钮
3. 填写表单:
   - **订货方*** (必填): 客户名称
   - **货物名称**: 商品名称
   - **货号**: 商品编号/货号
   - **编号**: 订单编号
   - **货物图片**: 粘贴图片URL或base64
   - **日期**: 订单日期
   - **数量**: 发货数量
   - **单价**: 单价
   - **件数**: 包装件数
   - **货物总价**: 商品总价
   - **运费**: 运输费用
   - **应付总价*** (必填): 客户应付总额
4. 点击 "保存"

**示例:**
- 订货方: 张三贸易公司
- 货物名称: T恤衫
- 货号: TS-001
- 编号: SO-20240215-001
- 数量: 500
- 单价: 25
- 件数: 10
- 货物总价: 12500
- 运费: 200
- 应付总价: 12700

### 2.2 上传货物图片

**方法1: 使用图片URL**
1. 将图片上传到图床或在线存储
2. 复制图片URL
3. 粘贴到 "货物图片" 字段

**方法2: 使用Base64编码**
1. 将图片转换为Base64编码
2. 粘贴到 "货物图片" 字段

**提示:** 在线工具可以帮助转换图片为Base64

### 2.3 添加付款记录

记录客户的付款。

**步骤:**
1. 展开订单(点击表格左侧的展开按钮)
2. 在 "付款记录" 区域,点击 "添加付款"
3. 填写表单:
   - **定金**: 预付定金金额
   - **已付金额**: 其他已付金额
   - **备注**: 付款说明
   - **日期**: 收款日期
4. 点击 "保存"

**示例:**
- 定金: 5000
- 已付金额: 0
- 备注: 首付定金
- 日期: 2024-02-15

**特殊说明:**
- 定金和已付金额都可以为负数(如退款)
- 可以添加多条付款记录
- 系统自动累计所有付款

### 2.4 查看欠款金额

系统自动计算客户欠款:

```
欠款金额 = 应付总价 - (所有定金之和 + 所有已付金额之和)
```

**示例计算:**
- 应付总价: ¥12,700
- 定金: ¥5,000
- 已付金额: ¥3,000
- **欠款金额: ¥4,700**

**付款状态:**
- 欠款金额 > 0: 显示 "未付清" (黄色标签)
- 欠款金额 ≤ 0: 显示 "已付清" (绿色标签)

### 2.5 筛选和搜索订单

**筛选条件:**
- **订货方**: 客户名称
- **货品名称/货号**: 搜索商品名称或货号
- **付款状态**: 
  - 已付清: 显示欠款≤0的订单
  - 未付清: 显示欠款>0的订单
- **关键词搜索**: 搜索任意字段
- **时间范围**: 订单日期范围

**搜索示例:**

**场景1: 查看某客户的所有订单**
- 订货方: 张三贸易公司
- 点击 "查询"

**场景2: 查看所有未付清的订单**
- 付款状态: 未付清
- 点击 "查询"

**场景3: 查找特定商品的订单**
- 货品名称/货号: T恤
- 点击 "查询"

### 2.6 导出数据

**步骤:**
1. (可选) 设置筛选条件
2. 点击 "导出Excel" 按钮
3. 导出当前显示的所有订单

**Excel 文件内容:**
- **订单汇总** 表: 所有订单的概览
- **详细信息** 表: 每个订单的完整信息和付款记录
- **客户汇总** 表: 按客户统计的订单和金额

### 2.7 编辑和删除订单

**编辑订单:**
1. 展开订单
2. 点击 "编辑订单" 按钮
3. 修改信息后保存

**删除付款记录:**
1. 展开订单
2. 在付款记录表格中点击 "删除"
3. 确认删除

**删除订单:**
1. 展开订单
2. 点击 "删除订单" 按钮
3. 确认删除

## 三、实用技巧

### 3.1 快速对账

**场景:** 月底需要对账,查看某工厂的应付款

**操作:**
1. 进入 "生产账单信息"
2. 工厂名称: 输入工厂名
3. 时间范围: 选择当月
4. 点击 "查询"
5. 查看顶部 "总欠款" 显示
6. 点击 "导出筛选结果" 打印或存档

### 3.2 查询材料供应商费用

**场景:** 查看某材料厂的所有费用

**操作:**
1. 进入 "生产账单信息"
2. 关键词搜索: 输入材料厂名称(如"信达")
3. 点击 "查询"
4. 系统显示所有相关订单
5. 导出数据查看详细

### 3.3 客户对账

**场景:** 查看某客户的欠款情况

**操作:**
1. 进入 "发货货款统计"
2. 订货方: 输入客户名称
3. 点击 "查询"
4. 查看顶部 "客户总欠款"
5. 展开各订单查看明细

### 3.4 统计分析

**材料成本分析:**
1. 生产账单 → 导出全部
2. 打开Excel → 查看 "材料汇总" 表
3. 按材料类型查看总支出

**客户销售分析:**
1. 发货货款 → 导出Excel
2. 打开Excel → 查看 "客户汇总" 表
3. 查看各客户的订单数量和金额

### 3.5 数据备份

**定期备份建议:**
1. 每月底导出全部数据
2. 保存Excel文件作为存档
3. 重要数据备份MongoDB数据库

**MongoDB 备份命令:**
```bash
mongodump --db order_management --out C:\backup\order_20240228
```

## 四、常见业务场景

### 场景1: 新接生产订单

1. 创建生产订单(填写工厂名称、总价)
2. 支付材料费用时,添加材料记录
3. 支付工厂预付款时,添加付款记录
4. 随时查看还需支付多少

### 场景2: 货物发给客户

1. 创建发货订单
2. 收到定金时,添加付款记录(定金栏)
3. 后续收款时,继续添加付款记录
4. 系统自动显示客户是否付清

### 场景3: 月末财务结算

1. 生产账单:
   - 按工厂筛选,导出各工厂的欠款
   - 按材料类型导出,查看各类材料支出
2. 发货货款:
   - 筛选"未付清"订单,催收欠款
   - 按客户导出,制作对账单

### 场景4: 材料退款

1. 展开相关订单
2. 添加材料记录
3. 价格填写负数(如: -500)
4. 备注说明退款原因
5. 系统自动调整应付金额

### 场景5: 客户退款

1. 展开发货订单
2. 添加付款记录
3. 已付金额填写负数(如: -1000)
4. 备注说明退款原因
5. 系统自动调整欠款金额

## 五、注意事项

1. **必填字段**: 带 * 的字段必须填写
2. **数据精度**: 金额自动保留两位小数
3. **负数处理**: 价格可以为负数,用于记录退款
4. **删除操作**: 删除后无法恢复,请谨慎操作
5. **数据安全**: 定期备份数据库和导出Excel
6. **网络要求**: 本地运行,不需要网络连接
7. **浏览器**: 推荐使用Chrome、Edge或Firefox
8. **并发使用**: 多人同时使用需要部署在服务器上

- **Ctrl + F**: 浏览器内搜索订单

- **Enter**: 在表单最后一个字段按Enter可快速保存

## 七、移动设备使用

系统基于响应式设计,但推荐在电脑上使用以获得最佳体验。

如需在平板或手机上使用:
1. 使用横屏模式
2. 表格内容可左右滚动查看
3. 某些功能可能需要放大页面
